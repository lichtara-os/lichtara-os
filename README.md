# Lichtara OS

Lichtara OS é um sistema operacional experimental.

## Instalação

1. Clone o repositório:
   ```sh
   git clone https://github.com/deboralutz/lichtara-os.git
   ```
2. Siga as instruções de instalação no arquivo `INSTALL.md`.
